Overview

Cloudera data lake is a service for launching and managing data lake clusters powered by Cloudera Runtime.

Version information

Version : 0.9.62 (BETA)

License information

License : Apache 2.0
Terms of service : https://www.cloudera.com/legal/commercial-terms-and-conditions.html

URI scheme

Schemes : HTTPS

Consumes

  • application/json

Produces

  • application/json
