Overview

Service for managing custom image catalogs and their associated Cloudera Runtime and FreeIPA images.

Version information

Version : 0.9.100

License information

License : Apache 2.0
Terms of service : https://www.cloudera.com/legal/commercial-terms-and-conditions.html

URI scheme

Schemes : HTTPS

Consumes

  • application/json

Produces

  • application/json
