Overview

Cloudera DataCatalog Service is a web service, using this service a user can execute operations like launching profilers in DataCatalog.

Version information

Version : 0.9.158

License information

License : Apache 2.0
Terms of service : null

URI scheme

Schemes : HTTPS

Consumes

  • application/json

Produces

  • application/json
