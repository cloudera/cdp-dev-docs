describe-service¶

subtitle

Describe Cloudera Data Engineering(CDE) Service

version

0.9.39

Description¶

Returns details of CDE service.

Synopsis¶

  describe-service
--cluster-id <value>
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--cluster-id (string) Cluster id of the service to be described.

--cli-input-json (string) Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean) Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.

Detailed description of a CDE service.

Name of the CDE Service.

CDP Environment Name.

CRN of the environment.

CDE tenant ID

Object to store resources for a CDE service.

Instance type of the CDE service.

Maximum instances for the CDE service.

Initial instances for the CDE service.

Maximum Number of Spot instances.

Initial Spot Instances for the CDE Service.

Root Volume Size.
