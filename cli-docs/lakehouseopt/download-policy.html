download-policy¶

subtitle

Downloads the policy definition for a particular policy version.

version

0.9.159

Description¶

Downloads the policy scripts and/or arguments for a particular policy version. Version is identified by a unique ID.

Synopsis¶

  download-policy
--policy-name <value>
--policy-version <value>
--datahub-crn <value>
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--policy-name (string)

The name of the policy.

--policy-version (string)

The policy version.

--datahub-crn (string)

The CRN of the CLO Data Hub.

--cli-input-json (string)

Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean)

Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.

Output¶

script -> (blob)

The policy script.

arguments -> (blob)

The policy arguments.

Form Factors¶

public
