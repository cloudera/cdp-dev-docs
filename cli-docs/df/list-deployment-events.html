list-deployment-events¶

subtitle

List event history for deployment.

version

0.9.97

Description¶

List event history for deployment.

Synopsis¶

  list-deployment-events
--deployment-crn <value>
[--rows <value>]
[--timestamp-from <value>]
[--filters <value>]
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--deployment-crn (string)

The deployment CRN

--rows (integer)

Max number of rows to return

--timestamp-from (integer)

Timestamp from when to list events prior

--filters (array)

Filter criteria

Syntax:

"string" "string" ...

--cli-input-json (string)

Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean)

Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.

Output¶

eventSummaries -> (array)

The events

item -> (object)

An event summary

crn -> (string)

The CRN of the event

name -> (string)

Name of the event

severity -> (string)

The severity of the event

firstOccurrence -> (integer)

Timestamp of the first occurrence of this event

eventType -> (string)

The type of event that occurred

alertType -> (string)

Optional value for alerts
