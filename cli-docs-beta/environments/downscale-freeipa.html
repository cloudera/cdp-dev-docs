downscale-freeipa¶

subtitle

Downscales FreeIPA instances.

version

0.9.77

Description¶

FreeIPA is an integrated Identity and Authentication solution that can be used for any of CM, CDP services.

Synopsis¶

  downscale-freeipa
--environment-name <value>
--target-availability-type <value>
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--environment-name (string)

The name or CRN of the environment.

--target-availability-type (string)

The target FreeIPA availability type.

Possible values:

  • TWO_NODE_BASED

--cli-input-json (string)

Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean)

Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.

Output¶

originalAvailabilityType -> (string)

The target FreeIPA availability type.

targetAvailabilityType -> (string)

The target FreeIPA availability type.

operationId -> (string)

UUID of the request for this operation. This ID can be used to get the status of the operation.

Form Factors¶

public, private
