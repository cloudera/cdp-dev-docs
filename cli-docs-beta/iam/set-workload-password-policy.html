set-workload-password-policy¶

subtitle

Set the workload password policy for the account.

version

0.9.36 (BETA)

Description¶

Set the workload password for the account. Changes to the workload password policy only affect passwords that are set after the policy has been updated. By default, passwords never expire.

Synopsis¶

  set-workload-password-policy
[--max-password-lifetime-days <value>]
[--global-password-policy <value>]
[--machine-users-password-policy <value>]
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--max-password-lifetime-days (integer) The maximum lifetime of passwords, in days. If set to ‘0’ passwords never expire.

--global-password-policy (object) An object representing password policy options.

Shorthand Syntax:

maxPasswordLifetimeDays=integer,minPasswordLength=integer,mustIncludeUpperCaseCharacters=boolean,mustIncludeLowerCaseCharacters=boolean,mustIncludeNumbers=boolean,mustIncludeSymbols=boolean

  "mustIncludeNumbers": true|false,

--machine-users-password-policy (object) An object representing password policy options.

Shorthand Syntax:
