get-environment¶

Description¶

Gets environment.

Synopsis¶

  get-environment
--crn <value>
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--crn (string) The environment CRN

--cli-input-json (string) Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean) Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.

Output¶

environment -> (object)

The DataFlow view of a CDP environment.

crn -> (string)

The environment CRN.

name -> (string)

The environment name.

cloudPlatform -> (string)

The cloud platform of the environment.

region -> (string)

The region of the environment.

deploymentCount -> (integer)

The deployment count.

minK8sNodeCount -> (integer)

The minimum number of Kubernetes nodes that need to be provisioned in the environment

maxK8sNodeCount -> (integer)

The maximum number of kubernetes nodes that environment may scale up under high-demand situations

status -> (object)

The status of a DataFlow enabled environment.

state -> (string)

The state of the environment.

message -> (string)

A status message for the environment.

k8sNodeCount -> (integer)

The number of kubernetes nodes currently in use by DataFlow for this environment.

instanceType -> (string)

The instance type of the kubernetes nodes currently in use by DataFlow for this environment.

dfLocalUrl -> (string)

The URL of the environment local DataFlow application.

authorizedIpRanges -> (array)

The authorized IP Ranges

(string)

activeWarningAlertCount -> (integer)

Current count of active alerts classified as a warning

activeErrorAlertCount -> (integer)

Current count of active alerts classified as an error

clusterId -> (string)

Cluster id of the environment, if enabled
