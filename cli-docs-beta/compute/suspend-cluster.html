suspend-cluster¶

subtitle

Suspend Cluster

version

0.9.135

Description¶

Suspends the compute cluster

Synopsis¶

  suspend-cluster
--cluster-crn <value>
[--skip-workloads-validation | --no-skip-workloads-validation]
[--skip-validation | --no-skip-validation]
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--cluster-crn (string)

Compute cluster CRN.

--skip-workloads-validation | --no-skip-workloads-validation (boolean)

Whether to skip the running workloads validation for externalized clusters

--skip-validation | --no-skip-validation (boolean)

Whether to skip validation.

--cli-input-json (string)

Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean)

Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.
