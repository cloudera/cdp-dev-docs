create-instance-groups¶

subtitle

Create instance groups

version

0.9.115

Description¶

Creates instance groups in a compute cluster.

Synopsis¶

  create-instance-groups
--cluster-crn <value>
--instance-groups <value>
--cluster-state-version <value>
[--skip-validation | --no-skip-validation]
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--cluster-crn (string)

Compute cluster CRN.

--instance-groups (array)

Instance groups to be created in the cluster.
