create-aws-access-key-credential¶

subtitle

Create an AWS Access Key cloud credential.

version

0.9.135

Description¶

Creates a new AWS Access Key cloud credential.

Synopsis¶

  create-aws-access-key-credential
--clusters <value>
--name <value>
--access-key <value>
--secret-key <value>
[--cli-input-json <value>]
[--generate-cli-skeleton]

Options¶

--clusters (array)

List of cluster CRNs where the credential should be created.

Syntax:

"string" "string" ...

--name (string)

Name of the credential.

--access-key (string)

AWS access key.

--secret-key (string)

AWS secret key.

--cli-input-json (string)

Performs service operation based on the JSON string provided. The JSON string follows the format provided by --generate-cli-skeleton. If other arguments are provided on the command line, the CLI values will override the JSON-provided values.

--generate-cli-skeleton (boolean)

Prints a sample input JSON to standard output. Note the specified operation is not run if this argument is specified. The sample input can be used as an argument for --cli-input-json.

Output¶

errors -> (array)

The list of response errors in case of a partial failure while creating the credential.

item -> (object)

API error.

code -> (integer)

Error code.

message -> (string)

Error message.

status -> (string)

Error status.

clusterUrl -> (string)

Cluster URL.

requestId -> (string)

Request ID.
